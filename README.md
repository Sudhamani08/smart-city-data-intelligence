Smart City Data Intelligence
This project analyzes traffic patterns and air quality trends using Python.
It includes data cleaning, exploratory data analysis, correlation analysis,
and data visualization to derive insights for smart city planning.

Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
  
 Project Structure
- raw/ → original datasets
- data/ → cleaned datasets
- notebooks/ → analysis notebooks
- processed/ → final processed outputs
